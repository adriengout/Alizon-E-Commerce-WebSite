# Projet Alizon - Plateforme E-commerce

Ce projet consiste en la conception et le développement d'une application web complète (Full Stack) dédiée à la vente en ligne, réalisée dans le cadre de mon BUT Informatique à l'IUT de Lannion.

## Présentation Générale
Alizon est une solution e-commerce robuste permettant la gestion d'un catalogue de produits, le tunnel d'achat client et une administration simplifiée via un outil dédié.

### Focus : Le Délivraptor
Le **Délivraptor** est l'interface de gestion (Back-Office) spécifique au projet. Il permet de piloter l'activité de la plateforme :
* **Gestion Logistique** : Suivi des commandes et lien avec le système de livraison.
* **Administration du Catalogue** : Gestion des stocks et mise à jour des produits.
* **Gestion des Données** : Administration des comptes clients et contrôle des flux de données.

## Stack Technique
Le projet s'appuie sur une architecture moderne pour assurer performance et sécurité :
* **Backend** : PHP et Python pour la logique métier.
* **Frontend** : Interfaces dynamiques conçues en HTML5, CSS3 et JavaScript.
* **Base de données** : Modélisation et exploitation complète sous **PostgreSQL**.
* **Versionnage** : Git & GitHub pour le suivi du développement et la collaboration.

## Fonctionnalités Clés
* **Conception Full Stack** : Développement complet du front-office et du back-office.
* **Modélisation de données** : Création d'un schéma relationnel complexe pour lier les utilisateurs, les commandes et les produits.
* **Expérience Utilisateur** : Interactions fluides grâce à JavaScript pour une navigation dynamique.

## Méthodologie de Travail
Le développement a été réalisé en suivant les principes de l'**Agilité** :
* **Collaboration** : Travail en équipe et communication technique constante.
* **Réactivité** : Adaptation aux contraintes techniques et gestion de projet coordonnée.
